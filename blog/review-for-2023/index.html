2023 年的個人極簡回顧與 2024 年展望

By Ping 31 January 2024

2024 年初，當我在社群媒體上看到許多人分享他們對 2023 年的回顧時，一股焦慮情緒湧上心頭。他們似乎都有所成就，而我卻感到自己未能達成大部分目標…

為了克服這樣的焦慮，我重新思考了年度回顧的主要目的是什麼？是為了幫助自己更清晰地設定下一年的目標。這種比較的心態並不健康，考慮到我在 2023 年職業生涯上經歷了重大轉變，我意識到當初設定的目標可能不再符合當下的需求。於是我總結出了幾個原則：

  1. 回顧帶來改善的方向： 回顧的目的在於幫助自己了解如何改進未來。
  2. 簡單有效的紀錄方式： 不需要繁瑣的流水帳式記錄，採用簡單直接的方式即可。
  3. 客觀看待自己的表現： 用第三者的角度來評價自己，認識到自己的優點與有待提升之處。

2023 年的成就

1. 自我約束與進步

控制自己不再任意購買線上課程和書籍。我成功地限制了自己在這方面的支出，更加專注地完成了之前購買的三門英文線上課程。

2. 專業發展

雖然最初並未計劃，但為了應對工作的需求，我考取了兩張金融證照。在面對新挑戰時，可以保持彈性，適時的新增新目標。

3. 志工經驗的收穫

我在下半年加入了 PyLadies 與 TWiDS，分別是與 Python 與 Data 有關的志工組織。這給我帶來了許多意想不到的收穫。從擔任助教到與同領域的人交流，我感受到了自己很享受當助教的過程，也很開心認識很多新朋友。

2023年的反思與改進

1. 寫作的挑戰

我發現自己在寫作方面並沒有達到預期的水準。雖然我會將靈感記錄下來，但由於完美主義，我往往無法將文章提升到我滿意的程度。然而，我意識到寫作最初的目的是為了分享，這將成為我未來努力的動力。

2. 專業技能的提升

在工作轉換後，我學習了許多新的技能，像是 Airflow 做資料工程, API 服務上線，MECE 拆解問題，儘管已往前了一大步，但我也意識到還有許多需要提升的地方。從 SQL 到 Python，我還在持續不斷地學習和精進，另一方面很重要的，是希望從專業領域中，去尋找自己在資料這一塊的定位到底是什麼。

2024年的展望

最後，我將我的未來願景總結為以下三點：

  1. 成為一名在資料專業領域能讓別人信任的人
  2. 積極分享所學所思，幫助他人成長
  3. 保持身心健康，並學會有效管理自己的生活。

透過這些願景，我再向下訂定 Q1 的計劃，並將這些目標具體化，以便每天都能向著自己的目標邁進。

結語

當我寫下這些回顧時，我發現心中的焦慮減緩了許多。這個過程讓我意識到，最終比較的對象應該是自己，而不是別人。透過回顧，我不僅可以認識到自己的成就和成長，也可以找到未來改進的方向。因此，我鼓勵每個人都動手寫下自己的2023回顧，不要害怕面對自己的不足，因為正是從這些反思中，我們才能不斷地成長和進步。
